# Expense Tracker
Expense Tracker is a simple web application built using Flask, HTML, CSS, and a touch of JavaScript. It allows users to track their expenses, display them in a table, and even remove expenses. The app also offers currency selection and formatting options.

# Features
Add expenses with item name, amount, and currency selection.
Display expenses in a table with detailed information.
Format expense amounts and total to always display with three decimal places.
Remove individual expenses from the list.
Technologies Used
Python (Flask) for backend logic.
HTML for creating the user interface.
CSS for styling the application.
JavaScript for interactive features.
Git for version control.
Getting Started
Clone the repository: git clone <repository-url>
Install dependencies: pip install -r requirements.txt
Run the app: python app.py
Access the app in your web browser at http://localhost:5000
How to Use
Add an expense by entering the item name, amount, and selecting a currency.
The expense will be displayed in the table below.
You can remove an expense by clicking the "Remove" button next to it.
